# 던전앤코파일럿광장
- "던전앤코파일럿"이라는 게이미피케이션 워크샵/캠페인을 위한 사용자 포털을 만듭니다.
-  이 포털을 이용해 사용자가 자신이 미션에서 수행한 결과물을 업로드할 수 있습니다.

## 기본 요구사항
- html 파일로 동작하는 웹앱을 만듭니다.
- 첫화면은 index.html 로 하겠습니다.
- 기능적으로 필요하면 별도의 html을 만들어서 링크로 연결해도 됩니다. 
- 자바스크립트와 CSS 는 별도의 파일로 분리하여 재사용 가능하도록 합니다.
- 깔끔하고 정돈된 디자인으로 만듭니다.
- 서버사이드 기능이 없이 자바스크립트만으로 동작하도록 기능을 구현해야 합니다.

## 저장소 관련 정의
- 사용자가 제출하는 파일이나 이미지는 Azure Blob Storage를 저장소로 사용합니다.
- 저장소에 엑세스 하기 위한 Key 등은 JS 파일에 저장하여 사용합니다.
- 이미지 제출이나 문저 제출과 같은 용도별 컨테이너를 분리하여 사용하도록 합니다.

## 기능 1. index.html 화면 구성
- 화면은 좌측에 카테고리 탭이 보여지고, 카테고리를 클릭하면 오른쪽에 그 카테고리와 관련한 화면이 보여지는 형식입니다.
- 카테고리는 나중에 추가될 수 있도록 자바스크립트의 전역변수 쪽에 정의합니다.
- 카테고리의 속성은 표시이름, ID, 종류(업로드, 갤러리뷰, 파일목록뷰, 이미지투표뷰, 이미지월드컵뷰), 연결컨테이너 로 이루어 집니다.
- 카테고리의 아이콘은 알아서 채워줍니다.

## 기능 2. 업로드
- 업로드 카테고리가 클릭되면 오른쪽에 버튼이 표시됩니다.
- 버튼을 클릭하면 파일을 업로드할 수 있습니다.
- 사용자가 선택한 파일이 이미지라면 미리보기로 업로드 하려는 이미지를 미리볼 수 있습니다.

## 기능 3. 갤러리뷰 의 화면 기능 정의
- 갤러리뷰 카테고리가 클릭되면 오른쪽에 갤러리 뷰 화면을 보여줍니다.
- 연결된 컨테이너의 저장된 이미지를 불러와 갤러리 뷰로 보여줍니다.
- 연결된 컨테이너에 있는 이미지가 아닌 다른 파일형식은 보여주지 않습니다.
- 갤러리 뷰에서 보여지는 보기를 "작게보기", "크게보기" 로 전환할 수 있습니다.
- 각각의 이미지를 클릭하면 모달레이어로 그 이미지를 크게 볼 수 있습니다.

## 기능 4. 파일목록뷰 의 화면 기능 정의
- 파일목록뷰 카테고리가 클릭되면 오른쪽에 파일목록 뷰 화면을 보여줍니다.
- 연결된 컨테이너의 모든 파일을 불러와 탐색기 처럼 보여줍니다.
- 각각의 파일을 클릭하면 모달레이어로 그 파일의 상세정보를 보여줍니다.

## 기능 5. 이미지투표뷰
- 이미지투표뷰가 클릭되면 오른쪽에 이미지 튜표 뷰 화면을 보여줍니다.
- 연결된 컨테이너에 저장된 이미지를 모두 보여주고 그중에 하나의 이미지에 투표할 수 있도록 합니다.
- 쿠키등을 이용하여 사용자는 하나의 이미지에만 투표하고 다시 방문하면 내가 투표한 이미지를 확인할 수 있습니다.
- 사용자가 투표한 기록은 연결된 컨테이너 안에 파일을 하나 만들어서 기록합니다.
- 한번 투표를 했더라도 다시 변경할 수 있고, 그 내용은 컨테이너 안의 파일에 반영됩니다.
- 이미지 마다 표를 받은 수를 배지형식으로 보여줍니다.

## 기능 6. 이미지월드컵뷰
- 이미지월드컵뷰가 클릭되면 오른쪽에 이미지 월드컵 화면을 보여줍니다.
- 연결된 컨테이너에 저장된 이미지를 대상으로 토너먼트 방식의 이미지 월드컵기능을 제공합니다.

## 추가요구사항
- 한번 투표한 사용자가 재투표를 할 수 없습니다.
- 한번 투표한 사용자가 다른 이미지를 클릭하면 투표권을 변경하는 것입니다.
- 이 사이트는 익명 접근이 가능하므로, 현재 사용자의 식별이 안됩니다.
- 따라서 현재 접속중인 브라우저의 정보등을 기반으로 사용자 식별을 하도록 합니다. 그리고 이 정보를 투표하기 기능에 적용하여 이미 투표를 한 사람이 또 투표하지 않도록 처리합니다.
- 이미지를 업로드 할 때, 업로드한 사용자를 식별 할 수 있게, 메일주소를 입력받는 기능을 추가해줘요. 이 이메일 주소는 갤러리나 투표, 월드컵 등의 뷰에서 보여줘야 합니다.
- index.html 뒤에 파라메터를 붙여서 해당 카테고리로 바로 이동할 수 있도록 합니다.
