분석가의 업무를 돕기 위해 M365 코파일럿에서 데이터 생성

  1. M365 코파일럿을 실행합니다
  2. 웹탭에서 아래의 프롬프트를 수행합니다. (대괄호 내용은 임의 수정 후 대괄호는 삭제)

데이터 생성 프롬프트

우리회사는 각종 제품을 생산하여 판매하고 있습니다. 그 중 [냉장고], [세탁기], [텔레비전], [에어콘]에 대한 실제 고객의 제품 피드백을 설문조사로 수집하였습니다.
- 제품의 종류와 구체적인 사용자의 피드백 내용을 목업데이터로 만들어 주세요. 
- 각 제품별로 30개의 피드백을 1개의 테이블로 생성합니다
- 모든 피드백은 50자 이상의 문장으로 작성합니다.
- 모든 피드백은 중복되지 않게 고유하게 작성합니다.
엑셀 파일로 다운로드 할 수 있게 해줘요.

분석가 에이전트로 데이터 분석

  1. 코파일럿이 만들어준 엑셀파일을 다운로드 합니다.
  2. 코파일럿에서 분석가 에이전트로 이동합니다.
  3. 다운로드한 엑셀 파일을 분석가 에이전트에 드래그앤 드롭합니다.
  4. 아래 프롬프트를 입력하고 실행합니다.
  5. 분석을 위해 적절한 질문(프롬프트)를 더 입력해 봅니다.

코파일럿의 GPT-5 기능으로 데이터 분석

  1. 코파일럿 챗으로 이동합니다
  2. 웹 탭에서 "GPT-5"를 켭니다
  3. 다운로드한 엑셀 파일을 코파일럿에 드래그앤 드롭합니다.
  4. 프롬프트를 입력하고 실행합니다.
  5. 분석을 위해 적절한 질문(프롬프트)를 더 입력해 봅니다.

GPT-5를 활용한 분석 프롬프트

피드백 내용을 해석하여 부정(0) 부터 긍정(10)까지 정량화 하여 분포를 1점 단위로 세분화해서 다시 시각화 해줘요. 
차트는 plotly를 이용해서 다시 그려줘요 (한글깨짐 해결). 
제품을 구분해서 분포도를 그려줘요
각 제품의 부정적인 피드백을 기반으로 개선해야할 사항을 도출하고, 주요 개선사항을 효과적으로 수행하기 위한 방안을 리서치 하여 정리해줘요
코파일럿이 제안하는 다음 단계 몇 개를 수행해 봅니다. (예: PoC 체크리스트 예시 보여줘)
